# Project Story

Hearing Gulangyu began as an accessibility and cultural-heritage navigation project for Gulangyu Island. The core question was:

How can a navigation system for a historic island consider not only distance, but also safety, sensory experience, slope, road texture, and different user abilities?

## Design Direction

The project combines three layers:

1. Spatial navigation logic
2. Unity digital twin simulation
3. Heritage and sensory storytelling

The app layer focuses on route guidance and user intent. The Unity layer tests hazard exposure, digital cane logic, movement, and route following. The research layer evaluates route quality and communicates results through charts.

## User Groups

The routing model considers multiple user profiles:

- general visitors
- elderly visitors
- wheelchair or mobility-impaired users
- visually impaired users

Each profile changes how slope, surface friction, and distance affect route choice.

## Outcome

The completed project demonstrates a workflow for building a spatial product around heritage navigation:

- collect and structure spatial route information
- model route weights
- simulate hazards and route following
- test user-profile behavior
- package results as a design and technology case study
